# United Hearts Website

Multi-page React site for United Hearts, built with Motion, Lucide, and a 21st.dev-compatible component structure.

Pages: Home, About Us, Get Involved, Partnerships, and Contact Us. The Home page uses a Farm Africa-inspired scroll story with cursor-responsive shapes, a zoom-through image moment, and concise mission sections.

## Run

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically build and publish the website to GitHub Pages at:

`https://unitedheartsdtx.github.io`

The Pages workflow also publishes `index.html` as `404.html` so direct visits and refreshes on client-side routes continue to work.

## Public contact configuration

Instagram is configured as `@unitedheartsdtx` in `src/data/site.js`. Contact submissions are delivered to `unitedheartsdtx@gmail.com` and copied to `subhas.angara@gmail.com` through FormSubmit.
